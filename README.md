# 🚀 API de Gerenciamento de Tarefas

Projeto desenvolvido com **ASP.NET Core Web API** como parte de um desafio técnico.

---

## 📌 Sobre

Esta API permite realizar operações básicas de gerenciamento de tarefas, como:

- Criar tarefas
- Listar tarefas
- Atualizar tarefas
- Remover tarefas

---

## 🛠️ Tecnologias utilizadas

- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server
- Swagger
